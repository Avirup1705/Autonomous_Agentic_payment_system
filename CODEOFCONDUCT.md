# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this project a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity, experience level, education, nationality, personal appearance, race, religion, or sexual identity.

We are committed to creating a welcoming, respectful, and inclusive environment for all contributors.

---

## Expected Behavior

Participants in this project community are expected to:

- Be respectful and considerate in communication.
- Use welcoming and inclusive language.
- Accept constructive criticism gracefully.
- Focus on what is best for the community and the project.
- Help create a positive and collaborative environment.

---

## Unacceptable Behavior

The following behaviors are considered unacceptable:

- Harassment, trolling, or insulting comments.
- Personal or political attacks.
- Publishing others’ private information without permission.
- Discriminatory language or behavior.
- Any conduct that could reasonably be considered inappropriate in a professional setting.

---

## Reporting Issues

If you experience or witness unacceptable behavior, please report it by contacting the project maintainers.

Reports can be submitted through:

- GitHub Issues
- Project maintainers via email
- Direct communication with repository administrators

All reports will be reviewed and handled confidentially.

---

## Enforcement

Project maintainers are responsible for clarifying and enforcing the standards of acceptable behavior.

Maintainers may remove, edit, or reject comments, commits, code, issues, or other contributions that violate this Code of Conduct.

Repeated violations may result in temporary or permanent bans from contributing to the project.
